# Commandes

Module OceanOS pour visualiser et traiter les commandes PrestaShop.

- Liste les commandes recentes via le connecteur PrestaShop configure dans OceanOS.
- Affiche le client, le paiement, les montants et les lignes produit.
- Permet de changer le statut PrestaShop en creant un `order_history`.

URL locale WAMP :

```text
/Commandes/
```
