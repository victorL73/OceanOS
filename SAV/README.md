# SAV

Module OceanOS pour traiter les demandes clients PrestaShop.

- Liste les `customer_threads` PrestaShop.
- Affiche les messages du fil client.
- Permet d'ajouter une reponse ou une note interne dans `customer_messages`.
- Permet de changer le statut du fil : ouvert, en attente ou ferme.

URL locale WAMP :

```text
/SAV/
```
