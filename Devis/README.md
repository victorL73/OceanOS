# Devis

Module OceanOS dedie a la creation de devis PDF.

- utilise la session OceanOS
- lit la configuration PrestaShop partagee dans OceanOS
- recupere les produits PrestaShop via le Webservice
- enregistre les devis dans `devis_quotes`
- genere des PDF dans le meme style que les devis Mobywork

URL locale :

```text
http://localhost/Devis/
```
